# Chatbot Project

This is a simple chatbot web application built using Python (Flask) and HTML.

## Features
- User can enter text
- Bot gives predefined responses
- Simple UI using HTML templates

## Files
- app.py → Main Python Flask file
- templates/index.html → UI File

## How to Run
1. Install requirements: `pip install flask`
2. Run the app: `python app.py`
3. Open browser: http://127.0.0.1:5000
